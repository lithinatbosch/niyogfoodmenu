# Icon Assets

This directory should contain app icons in the following sizes:
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

## How to Generate Icons

You can use online tools or design software to create these icons:

1. **Design the icon** (512x512 is recommended as the base size)
   - Use a food/meal-related emoji or symbol
   - Background: Light green (#8FBC8F)
   - Simple and recognizable design

2. **Resize to all required dimensions**
   - Use tools like:
     - https://www.favicon-generator.org/
     - https://realfavicongenerator.net/
     - Photoshop, GIMP, or other image editors

3. **Save as PNG** with transparency if desired

## Quick Solution

For quick setup, you can:
1. Use an emoji as an icon (🍽️, 🥗, 📅)
2. Convert emoji to PNG using online tools
3. Resize to all required dimensions

## Temporary Solution

If you need placeholder icons immediately, you can create simple colored squares with the app name using any image editor.
