# Changelog

All notable changes to Kids Menu Planner will be documented in this file.

## [1.0.0] - 2026-06-15

### Initial Release

#### Features
- ✅ Dashboard showing today's menu
- ✅ Food items management (CRUD operations)
- ✅ Weekly meal planner for 7 days
- ✅ Calendar view of weekly menu
- ✅ Copy previous week functionality
- ✅ Random menu suggestions
- ✅ Search and filter food items
- ✅ Progressive Web App (PWA) support
- ✅ Offline functionality
- ✅ Mobile-first responsive design
- ✅ iOS Safari compatibility

#### Categories
- 🍳 Breakfast
- 🍱 Lunch
- 🍎 Snacks (for morning and evening)

#### Technical Features
- PHP 8.x with PDO
- MariaDB/MySQL database
- Vanilla JavaScript (no frameworks)
- CSRF protection
- Prepared statements for SQL injection prevention
- Input validation and sanitization
- Output escaping
- Session security
- Service Worker for offline caching
- Web App Manifest
- RESTful API endpoints

#### Design
- iOS Human Interface Guidelines inspired
- Soft pastel color scheme
- Light green (#8FBC8F) primary color
- Rounded cards and buttons
- Large touch-friendly controls
- Smooth animations
- Bottom navigation bar
- Modern typography

#### Sample Data
- 6 breakfast items
- 6 lunch items
- 8 snack items

#### Documentation
- Complete installation guide
- Nginx configuration
- Apache configuration (.htaccess)
- Quick start guide
- Troubleshooting section
- Security best practices

#### Browser Support
- Safari (iOS/iPadOS)
- Chrome (Desktop/Mobile)
- Firefox (Desktop/Mobile)
- Edge (Desktop/Mobile)

#### Server Support
- Nginx (recommended)
- Apache with mod_rewrite
